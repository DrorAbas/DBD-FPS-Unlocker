# Changelog

All notable changes to this project will be documented here.

## [1.0.0] - 2026-07-20

### Added

- First public release under the name **DBD FPS Unlocker**
- FPS selection from 30 to 360
- Standard and experimental methods
- Automatic config-folder detection
- Backup, verification and rollback
- Restore latest backup
- Optional Read-only protection
- V-Sync selection
- Delayed cursor tooltips
- Multilingual interface
- Dark red visual theme
- Explicit AI-development disclosure

### Compatibility

- Imports settings from earlier private `DBD FPS Tool` builds when available
- Can restore backups created by earlier private builds
- Removes configuration blocks produced by earlier private versions
