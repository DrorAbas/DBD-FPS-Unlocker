# AI Disclosure

DBD FPS Unlocker was created with substantial assistance from artificial intelligence.

## Tools used

AI assistance included OpenAI's ChatGPT for:

- PowerShell and WPF code generation
- Refactoring and debugging
- Interface wording and translations
- Documentation structure
- Static validation scripts
- Naming and release preparation

## Human involvement

The project owner:

- Defined the product requirements
- Directed the visual design
- Selected features and safety behavior
- Tested multiple builds on Windows
- Reported failures and requested corrections
- Approved the public-project direction

## Important limitation

AI-generated code can be incorrect, insecure or incomplete. Public source availability is not a guarantee of safety.

Users and contributors should review the code and test changes carefully. Security reports are welcome through the process described in `SECURITY.md`.

## Commit and contribution policy

AI-assisted contributions are allowed, but contributors should:

1. Disclose meaningful AI involvement.
2. Review every generated change.
3. Explain what was tested.
4. Avoid claiming untested generated code is verified.
